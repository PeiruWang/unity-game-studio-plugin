## Upstream Inspiration

This project is inspired by and adapted from OpenAI's Game Studio plugin:

- Project: `game-studio`
- Repository: https://github.com/openai/plugins/tree/main/plugins/game-studio
- Manifest: https://raw.githubusercontent.com/openai/plugins/main/plugins/game-studio/.codex-plugin/plugin.json
- Upstream license identifier in the plugin manifest: MIT

The OpenAI Game Studio plugin focuses on browser-game workflows. This repository adapts the general workflow-routing concept for Unity-focused development, including Unity 6, C#, scenes, prefabs, ScriptableObjects, uGUI, Unity asset pipelines, and Unity Test Framework verification.
